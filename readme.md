# Algoritmo Genético - Problema da Mochila (Knapsack Problem)

Esta aplicação resolve o clássico problema da mochila em um cenário de sobrevivência, utilizando um Algoritmo Genético customizado com Backend em Python e um Frontend em HTML/Javascript.

## Estrutura do Projeto
- `app.py`: Backend (API construída com Flask) que contém a lógica pesada do Algoritmo Genético.
- `index.html`: Frontend construído com HTML puro, CSS e JavaScript. Exibe a tela de configurações, itens, resultados e o gráfico de evolução utilizando a biblioteca `Chart.js`.
- `README.md`: Instruções detalhadas.

## Como executar

### 1. Requisitos Prévios
Certifique-se de ter o [Python](https://www.python.org/) instalado em sua máquina.

### 2. Instalação das dependências
Abra o terminal na pasta raiz do projeto e instale o Flask e a biblioteca de permissões de origem cruzada (CORS):

```bash
pip install flask flask-cors
```
